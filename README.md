# Quartz Test Bed

A sandbox repository for developing and testing Quartz themes, layouts, components, workflows, and site architecture.

## Live Site

https://solvesaint.github.io/quartz-test-bed/

## Purpose

This repository is not a content project.

It exists as a development environment for:

- Quartz customization
- Theme development
- CSS and SCSS experimentation
- Custom components
- GitHub Pages deployment testing
- Navigation and explorer improvements
- Reusable Quartz templates

## Current Theme Goals

The current test-bed theme is evolving toward:

- Dark forest aesthetic
- Layered SVG backgrounds
- Roaming silhouette animations
- Block-style explorer navigation
- Zebra striping
- Explorer chevrons
- TOC chevrons
- Strong readability in dark mode

## Quartz Features Being Evaluated

- Custom page headers
- Custom SVG assets
- Explorer enhancements
- Responsive layouts
- Graph view integration
- RSS generation
- Search indexing
- GitHub Actions deployment

## Deployment

The site is built automatically using GitHub Actions.

Build pipeline:

1. Install dependencies
2. Build Quartz
3. Generate static site output
4. Publish to GitHub Pages

## Repository Role

This repository serves as a staging environment for future Quartz-based sites.

Successful themes, components, workflows, and styling experiments can later be migrated into production repositories.
